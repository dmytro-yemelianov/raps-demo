Object Storage Service

4 workflows

Terminal — raps oss

Bucket & Object Lifecycle

Create buckets, upload files, list objects, generate signed URLs, and download files. Demonstrates the full object storage lifecycle.

$ raps bucket list
$ raps bucket create --key demo-bucket
$ raps object upload demo-bucket model.rvt
$ raps object list demo-bucket

Model Derivative

3 workflows

Terminal — raps translate

Translation Pipeline

Translate CAD models to SVF2, OBJ, STEP, and other formats. Monitor translation progress and download derivatives for offline use.

$ raps translate start $URN --format svf2
$ raps translate status $URN --wait
$ raps translate manifest $URN
$ raps translate download $URN -o ./derivatives
